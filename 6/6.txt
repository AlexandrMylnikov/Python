Исходные параметры по данным
Age : Возраст
Sex : Пол
exang: стенокардия, вызванная физической нагрузкой(1 = yes; 0 = no)
ca: количество крупных сосудов (0-3)
cp : Тип боли в груди
	Value 1: типичная стенокардия
	Value 2: атипичная стенокардия
	Value 3: боль, не связанная со стенокардией
	Value 4: бессимптомная
trtbps : кровяное давление в состоянии покоя (мм ртутного столба) - 
chol : уровень холестерина в мг/дл определяется с помощью датчика ИМТ -
fbs : (уровень сахара в крови натощак > 120 мг/дл)  (1 = true; 0 = false) -
rest_ecg : результаты электрокардиографии в состоянии покоя - 
	Value 0: Норма
	Value 1: Наличие аномалии зубца ST-T (инверсия зубца T и/или подъем или понижение ST > 0,05 мВ)
	Value 2: Наличие вероятной или определенной гипертрофии левого желудочка по критериям Эстеса
thalach : максимальная частота сердечных сокращений -
target : 0 = меньше шансов на сердечный приступ 1 = больше шансов на сердечный приступ

Из них неизвестны, но приведены в данных - oldpeak, slp, thall

По графику age совмещённым с показателю максимальная частота сердечных сокращений (thalach) видно, что до 55-лет количество людей с заболеванием больше чем без неё.
Аналогичная ситуация с параметрами: chol, trtbps, fbs, rest_ecg 

По диаграмме age колличество людей с заболеванием значительно уменьшилось, 
Как и до этого развитие заболевания стенокардия - совокупность множества факторов, где нельзя чётко сказать, что только один из них повлиял больше всего
